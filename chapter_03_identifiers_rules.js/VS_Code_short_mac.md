# VS Code Shortcuts — Mac

| Action | Shortcut |
|--------|----------|
| Command Palette | `Cmd+Shift+P` |
| Quick Open (File) | `Cmd+P` |
| Sidebar toggle | `Cmd+B` |
| Integrated terminal | `` Ctrl+` `` |
| New file | `Cmd+N` |
| Save | `Cmd+S` |
| Save all | `Cmd+Option+S` |
| Close tab | `Cmd+W` |
| Cut line | `Cmd+X` (no selection) |
| Copy line | `Cmd+C` (no selection) |
| Move line up/down | `Option+↑/↓` |
| Duplicate line | `Shift+Option+↑/↓` |
| Find | `Cmd+F` |
| Find and replace | `Cmd+Option+F` |
| Multi-cursor add | `Option+Click` |
| Toggle comment | `Cmd+/` |
| Block comment | `Shift+Option+A` |
| Format document | `Shift+Option+F` |
| Indent | `Cmd+]` |
| Outdent | `Cmd+[` |
| Go to definition | `F12` |
| Peek definition | `Option+F12` |
| Split editor | `Cmd+\` |
| Zoom in | `Cmd+=` |
| Zoom out | `Cmd+-` |
| Open settings | `Cmd+,` |
| Open keyboard shortcuts | `Cmd+K Cmd+S` |
