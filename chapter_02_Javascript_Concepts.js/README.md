## 05_JS_Basics.js

This script shows the fundamental steps of using a variable:

1. Declare a variable `v` using `var` and initialize it to `10`.
2. Print the current value using `console.log`.
3. Reassign the variable to a new value (`12`).
4. Print the updated value.

```javascript
var v = 10;
console.log(v);

v = 12;
console.log(v); 
```
